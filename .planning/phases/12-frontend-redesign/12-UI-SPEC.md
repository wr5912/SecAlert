# Phase 12 — UI Design Contract

> Visual and interaction contract for Phase 12: 前端视觉升级 (Frontend Visual Upgrade). Generated by gsd-ui-researcher.

---

## Design Direction

**Concept: "Tactical Command Center"**

SecAlert 是安全监控产品，目标用户是非专业运维人员。Phase 12 的视觉升级需要传达：
- **Purpose**: 将海量告警过滤为真正需要关注的威胁
- **Tone**: 专业、可信、严肃但不失现代感 —— 如同军事级别的指挥中心
- **Differentiation**: 深色主题 + 霓虹高光 + 几何切割 + HUD风格 —— 让安全监控界面有独特的记忆点

**美学原则:**
- 拒绝 Inter/Roboto/Arial —— 使用独特有个性的字体组合
- 大胆使用主色 + 尖锐 accent —— 安全产品常用深色主题
- 动效: CSS 优先，交错动画
- 布局: 不对称、网格打破、HUD 风格
- 背景: 渐变网格、噪点纹理、几何图案

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn |
| Preset | base-nova |
| Component library | radix (base-nova components) |
| Icon library | lucide |
| Technical font | JetBrains Mono (timestamps, IPs, IOC, logs) |
| Heading font | Space Grotesk (headings, labels) |
| Body font | IBM Plex Sans (body text, descriptions) |

**Font rationale:**
- JetBrains Mono: 清晰易读的等宽字体，适合展示 IP、时间戳、IOC 等技术数据
- Space Grotesk: 现代几何无衬线，有科技感和独特个性，适合安全产品标题
- IBM Plex Sans: 专业可靠，阅读舒适，适合正文

## Color

**Base Palette:**

| Role | Value | Usage |
|------|-------|-------|
| Background (dominant 60%) | #0a0f1a | Page background — 比 Phase 9 更深的藏青色 |
| Surface (secondary 30%) | #111827 | Cards, sidebar, panels |
| Border | #1e293b | Subtle borders |
| Accent (10%) | #00f0ff | Interactive elements, primary actions — Electric cyan |
| Accent muted | #00f0ff/10 | Accent backgrounds |

**Accent reserved for:**
- Selected nav item indicator (left border + glow)
- Primary button background
- Focus ring
- AI copilot accent line
- Links
- Active states

**Severity Colors:**

| Level | Value | Usage |
|-------|-------|-------|
| Critical | #ff2d55 | Alert badge, critical indicator — 霓虹红 |
| High | #ff6b35 | Alert badge, high indicator — 霓虹橙 |
| Medium | #fbbf24 | Alert badge, medium indicator — 琥珀黄 |
| Low | #4b5563 | Alert badge, low indicator — 暗灰 |

**Semantic Colors:**

| Role | Value | Usage |
|------|-------|-------|
| Success | #10b981 | 成功状态 |
| Warning | #f59e0b | 警告状态 |
| Destructive | #ff2d55 | 危险操作（删除等） |
| Info | #00f0ff | 信息提示 |

**Visual Effects:**

- **Card glow**: Cards with critical/high alerts get subtle glow effect using box-shadow with severity color at 20% opacity
- **Grid background**: Page background has subtle grid pattern (1px lines at #1e293b, 40px spacing)
- **Noise texture**: Overlay noise at 2% opacity for depth
- **Corner accents**: Cards have small geometric corner marks (HUD style)

## Layout Structure

```
+--------------------------------------------------+
|  Header (fixed, 56px)                            |
|  [Logo] [Nav Items...]              [AI Copilot] |
+--------+----------------------------------+------+
|        |                                  |      |
| Left   |  Main Content                    | Right|
| Nav    |  (fluid, max-w-7xl)              | Panel|
| 64px   |                                  | 320px|
| col-   |                                  |      |
| lapsed |                                  | AI   |
| 240px  |                                  | Panel|
| expand |                                  |      |
|        |                                  |      |
+--------+----------------------------------+------+
```

**Left Navigation:**
- Width: 64px collapsed (icon-only), 240px expanded
- Collapsed shows icons only with tooltip labels
- Expanded shows icons + labels
- Active item: left border accent + subtle glow
- Hover: background lighten + icon color shift

**Header:**
- Fixed height 56px
- Logo + nav items + AI copilot trigger
- Subtle bottom border with gradient fade

**Main Content:**
- Max width: 1280px (max-w-7xl)
- Fluid with responsive breakpoints
- Page title + content area

**Right Panel (AI Copilot):**
- Fixed 320px width
- Collapsible
- Accent top border

**Visual Pacing:**
- Dashboard: Dense grid of stat cards
- Alert List: Clean table-like layout with clear hierarchy
- Alert Detail: Two-column layout (content + context)
- Analysis Workbench: Three-column (nav + canvas + copilot)

**视觉焦点:** 仪表盘头部统计卡片区域，用户打开页面后视线应首先落在此处。统计卡片以加粗的数字和状态色彩传达核心安全态势。
