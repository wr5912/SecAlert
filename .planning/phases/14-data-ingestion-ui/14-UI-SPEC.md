## Interaction Mode

### A. 分步骤模态对话框向导 (Modal Wizard)

**触发方式:** 页面上的"添加数据源"或"接入新设备"按钮打开模态对话框。

**4个步骤:**

| Step | Title | Content |
|------|-------|---------|
| 1 | 选择设备类型 | 网格布局展示设备类型卡片（防火墙、入侵检测、VPN、交换机、路由器、其他） |
| 2 | 配置连接参数 | 表单：IP地址/主机名、端口、用户名、密码/密钥、连接协议 |
| 3 | 选择日志格式 | 日志格式选择（CEF、Syslog、JSON、自定义正则）+ 预览区域 |
| 4 | 完成 | 成功状态 + 摘要信息 + "完成"按钮 |

**Step Indicator (步骤指示器):**
- 水平排列，4个圆点/数字
- 当前步骤: accent color + 标签文字
- 已完成步骤: accent color + 勾选图标
- 未完成步骤: #4b5563 (Low severity gray) + 标签文字
- 步骤间有连接线

**Modal结构:**
- Header: Step标题 + 关闭按钮(X)
- Body: 当前步骤内容
- Footer: 取消按钮(左侧) + 返回/下一步按钮(右侧)

**Button States:**
- "上一步": ghost button, 仅Step 2-4显示
- "下一步": primary button (accent), 已在Step 1-3显示
- "完成": primary button (accent), 仅Step 4显示
- "取消": ghost button, 所有步骤显示

### B. 独立模板管理页面

**路由:** `/settings/data-sources` 或 `/ingestion/templates`

**页面结构:**
- Header: 页面标题 + "新建模板"按钮
- 内容区: 模板列表 (卡片布局)
- 每个模板卡片: 模板名称、设备类型图标、日志格式标签、操作按钮(编辑、删除)

**Template Card:**
- 尺寸: 宽度100%，最小高度 120px
- 布局: 左侧设备类型图标 + 中间模板名称/类型 + 右侧操作按钮
- 悬停: 边框变为accent color
