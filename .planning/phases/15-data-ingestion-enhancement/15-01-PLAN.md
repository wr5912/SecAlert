**OCSF 统一字段映射说明（参考 OCSF 调研文档）：**
- 所有异构日志（CEF/Syslog/JSON/Custom）必须归一化到 OCSF 格式
- OCSF 核心对象：`src_endpoint`（源端）、`dst_endpoint`（目的端）、`observables`（可观察对象数组）
- OCSF Categories：
  - Category UID 2: Findings（安全告警）
  - Category UID 1: System Activity（终端行为）
  - Category UID 4: Network Activity（网络行为）

**CEF → OCSF 映射规范：**
| CEF 字段 | OCSF 字段 |
|----------|-----------|
| src | src_endpoint.ip |
| spt | src_endpoint.port |
| dst | dst_endpoint.ip |
| dpt | dst_endpoint.port |
| proto | connection_info.protocol_num |
| msg | message |
| sev | severity_id |
| cn1 | unmapped.cn1（保留到 unmapped 对象）|
| 原始日志 | raw_data |

**OCSF 关键字段：**
- `type_uid = class_uid * 100 + activity_id`（如 200101 = Security Finding + activity 1）
- `observables`：提取 IOC（IP、域名、Hash）用于快速威胁情报匹配
- `time`：Unix Epoch 毫秒级时间戳
- `unmapped`：厂商特有字段放入此对象，原始日志保留在 raw_data

```python
class LogFormatRecognition(Signature):
    """识别日志格式并归一化到 OCSF"""
    raw_logs = InputField(desc="3-5条原始日志示例，每条一行")
    source_type = InputField(desc="数据源类型描述，如 firewall、suricata")

    detected_format = OutputField(desc="检测到的格式：CEF/Syslog/JSON/Custom")
    regex_pattern = OutputField(desc="Python 正则表达式，包含命名捕获组")
    ocsf_field_mappings = OutputField(desc="OCSF 归一化字段映射 JSON，如 {\"src_endpoint.ip\": \"src\", \"dst_endpoint.ip\": \"dst\", \"severity_id\": \"sev\", \"time\": \"timestamp\"}")
    category_uid = OutputField(desc="OCSF Category UID: 2 (Findings) / 1 (System) / 4 (Network)")
    class_uid = OutputField(desc="OCSF Class UID: 2001 (Security Finding) / 4001 (Network Activity)")
    confidence = OutputField(desc="置信度 0.0-1.0")
    reasoning = OutputField(desc="识别理由和 OCSF 归一化说明")
```
