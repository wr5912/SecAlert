# Phase 18 Plan 01: 多源异构数据模拟器基础设施

## 执行目标

创建独立于主系统的 `docker-compose-simulators.yml`，包含覆盖各种数据上报渠道的模拟服务。

## 需求分析

### 目标渠道
| 渠道 | 协议/方式 | 现有开源方案 |
|------|-----------|--------------|
| REST API 轮询 | HTTP GET | FastAPI 自建 |
| REST/Webhook 推送 | HTTP POST | summved/log-generator |
| Syslog 网络接收 | UDP/TCP 514 | syslog-ng / 自建 |
| 数据库/ES 轮询 | JDBC/ES REST | 自建 poller |
| 日志文件监控 | file tail | Vector / 自建 |

### 决策
- **summved/log-generator** 是最完整的开源 SIEM 日志生成器，支持 MITRE ATT&CK、12+ 数据源、多种输出格式（JSON/Syslog/CEF/HTTP）
- 补充其他渠道的专用模拟器
- 所有模拟器放在 `simulators/` 目录

## 任务分解

### Task 1: 创建 simulators 目录结构
**文件:**
- `simulators/docker-compose-simulators.yml`
- `simulators/.env.simulators.example`
- `simulators/README.md`

### Task 2: 配置 summved/log-generator 服务
**环境变量:**
- `SIEM_HTTP_URL=http://localhost:9000/api/v1/ingest` (SecAlert ingest endpoint)
- `ENABLE_SYSMGMT=true`
- `ENABLE_MONITORING=true`

**输出格式:** JSON + HTTP POST 到 SecAlert

### Task 3: 创建 Syslog 模拟器服务
**实现方式:** Python socket server
- 监听 UDP 514 接收 syslog
- 同时主动生成 CEF 格式 syslog 并发送
- 模拟防火墙、IDS 等设备

**配置文件:** `simulators/syslog-simulator/config.yaml`

### Task 4: 创建 REST API Polling 模拟器
**实现方式:** FastAPI
- 模拟 WAF 数据库表
- `GET /api/waf-logs` 返回 JSON 日志
- SecAlert 定时轮询获取

**端口:** 8081

### Task 5: 创建 Webhook 推送模拟器
**实现方式:** 复用 log-generator 的 HTTP 推送
- 配置 `SIEM_HTTP_URL` 指向 SecAlert
- 生成多种格式日志 (CEF, JSON)
- 实时推送

### Task 6: 创建日志文件模拟器
**实现方式:** Python file writer + Vector tail
- 模拟应用日志文件写入
- Vector 监控文件变化并发送到 Kafka

**配置:** `simulators/file-simulator/logs/app.log`

## 技术架构

```
simulators/
├── docker-compose-simulators.yml
├── .env.simulators.example
├── log-generator/           # summved/log-generator
│   └── config/
├── syslog-simulator/        # Python Syslog 服务
│   ├── server.py
│   └── config.yaml
├── rest-polling-simulator/  # FastAPI REST Polling
│   ├── app.py
│   └── config.yaml
└── file-simulator/          # 日志文件生成
    ├── writer.py
    ├── logs/
    └── vector.yml
```

## 验证

```bash
docker-compose -f simulators/docker-compose-simulators.yml up -d

# 验证 log-generator
curl http://localhost:3000/health

# 验证 REST polling
curl http://localhost:8081/api/waf-logs

# 验证 Syslog 接收
nc -u localhost 514

# 验证 SecAlert 收到数据
# 检查 Kafka topic: raw-events
```
