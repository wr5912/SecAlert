# Phase 9 — UI Design Contract

> Visual and interaction contract for Phase 9: 智能分析工作台 (Intelligent Analysis Workbench). Generated by gsd-ui-researcher.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn |
| Preset | base-nova |
| Component library | radix (base-nova components) |
| Icon library | lucide |
| Font | font-sans (UI/body), font-mono (technical data: timestamps, IOC, logs) |

**Note:** The `font-mono` class is used for technical data. UI text uses the default font-sans.

| Token | Value | Usage |

## Layout Structure

```
+------------------+---------------------------+------------------+
|                  |                           |                  |
|  Left Nav        |  Center Canvas            |  AI Copilot      |
|  (200px fixed)   |  (flex-1)                 |  (320px fixed)   |
|                  |                           |                  |
|  - 告警中心       |  [Attack Graph /          |  Investigation   |
|  - 威胁狩猎       |   Timeline /              |  Copilot         |
|  - 溯源时间线     |   Hunting Workbench]      |                  |
|  - 资产图谱       |                           |  - Query input   |
|                  |                           |  - Recommendations|
+------------------+---------------------------+------------------+
```

**Left Nav:** Fixed 200px, collapsible to 64px icon-only mode
**Center Canvas:** Flexible, adapts to selected analysis mode
**AI Copilot:** Fixed 320px, collapsible

**视觉焦点：** 告警中心（左侧聚类面板）默认为焦点区域，用户打开工作台后视线应首先落在此处。

**Accessibility：** 侧边栏折叠至64px图标模式时，NavItem 组件使用 `aria-label` 提供导航标签替代文字。
